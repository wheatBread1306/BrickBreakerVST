# Contributing to Brick Breaker Plugin

## 日本語 / Japanese

このジョークプラグインへの貢献に興味を持っていただきありがとうございます！🎮

### 貢献方法

このプロジェクトはエンターテイメント目的ですが、改善案やバグ報告は歓迎します。

#### バグ報告

バグを見つけた場合は、以下の情報を含めてIssueを作成してください：

- **OS**: Windows 10/11, macOS バージョン
- **DAW**: 使用しているDAWとバージョン
- **MIDIコントローラー**: 使用しているMIDIキーボード/コントローラー
- **問題の詳細**: 何が起きたか、期待される動作
- **再現手順**: 問題を再現する手順

#### 機能提案

新機能のアイデアがある場合：

1. まずIssueで提案を共有してください
2. 実装前にフィードバックを待つことをお勧めします
3. ジョークプラグインの精神を保ちつつ、楽しい機能を歓迎します！

#### プルリクエスト

コードの改善を提案する場合：

1. このリポジトリをフォーク
2. 新しいブランチを作成 (`git checkout -b feature/amazing-feature`)
3. 変更をコミット (`git commit -m 'Add some amazing feature'`)
4. ブランチにプッシュ (`git push origin feature/amazing-feature`)
5. プルリクエストを作成

**コーディング規約:**
- JUCEのコーディングスタイルに従う
- 既存のコードスタイルを維持
- 日本語と英語の両方でコメントを追加（可能であれば）

## English

Thank you for your interest in contributing to this joke plugin! 🎮

### How to Contribute

While this is an entertainment project, we welcome improvements and bug reports.

#### Reporting Bugs

If you find a bug, please create an Issue with:

- **OS**: Windows 10/11, macOS version
- **DAW**: Which DAW and version you're using
- **MIDI Controller**: Your MIDI keyboard/controller model
- **Issue Details**: What happened vs. what you expected
- **Steps to Reproduce**: How to recreate the issue

#### Feature Suggestions

If you have ideas for new features:

1. Share your proposal in an Issue first
2. Wait for feedback before implementing
3. Keep it fun and in the spirit of a joke plugin!

#### Pull Requests

To propose code improvements:

1. Fork this repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add some amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

**Coding Standards:**
- Follow JUCE coding style
- Maintain consistency with existing code
- Add comments in both Japanese and English (if possible)
